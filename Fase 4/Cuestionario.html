Responda las siguientes preguntas

1. ¿Cuál de los siguientes selectores CSS seleccionará todos los elementos de párrafo (p) en un documento HTML?

a. p { }
b. .p { }
c. #p { }

2. ¿Cuál de las siguientes opciones selecciona un elemento con el atributo "href" igual a "https://www.ejemplo.com" utilizando CSS?

a. [href = "https://www.ejemplo.com"] { }
b. a href="https://www.ejemplo.com" { }
c. a[href="https://www.ejemplo.com"] { }

3. ¿Cuál de las siguientes opciones seleccionará todos los elementos con la clase "boton" en un documento HTML?

a. boton { }
b. .boton { }
c. button { }

4. ¿Cuál de las siguientes opciones selecciona todos los elementos a que se encuentran dentro de un elemento div?

a. div a { }
b. a > div { }
c. a > div { }

5. ¿Cuál de las siguientes opciones selecciona el primer elemento li dentro de una lista desordenada (ul)?

a. ul li:first-child { }
b. ul li:first-of-type { }
c. ul:first-child li { }
